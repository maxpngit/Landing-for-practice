Одностраничник, построенный на принципе блочной верстки, по классической схеме: HTML + CSS + JS.

Не полностью реализована адаптивность:
 - все блоки должны быть по центру вне зависимости от разрешения монитора, зума и статуса окна браузера (развернуто на весь экран или частично).
 - должно быть соответствие требованиям представления на портативных устройствах (планшеты, смартфоны) в части расстояния между активными элементами и читабельности текстовой части.
 
 Клиентский JS-скрипт (feedback.js) призван визуализировать отзывы в режиме слайд-шоу. Сделано костыльно.
 
 Отсутствует форма обратной связи.
 
 Примечание: данный сайт не имеет большого практического значения, но может выступать как полигон для оттачивания приемов и современных принципов FrontEnd разработки.
